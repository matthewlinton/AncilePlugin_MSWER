Disable Microsoft Windows Error Reporting

ABOUT
Modify the registry to disable Microsoft Windows Error Reporting

	
CONFIGURATION
The following options can be added to config.ini

	DISABLEMSWER (Boolean) - Enable or disable the disable_MSWER plugin
		Y	- Enable the disable_MSWER Plugin (DEFAULT).
		N	- Disable the disable_MSWER Plugin.

		
NOTE
Disabling this plugin does not undo the changes made in previous runs.
